# Projeto Final - Introdução à Programação Orientada a Objetos (2025/2)
Este repositório contém o projeto final da disciplina de Introdução à Programação Orientada a Objetos, desenvolvido em meu segundo semestre no curso de Sistemas de Informação na UFLA.

## Sobre o projeto
O projeto, intitulado *"A Maldição de VilaVelha"*, é um jogo desenvolvido em Java utilizando o ambiente Greenfoot como parte prática da avaliação da disciplina de IPOO. O objetivo principal foi aplicar os conceitos fundamentais da Programação Orientada a Objetos, tais como **classes**, **objetos**, **métodos**, **encapsulamento**, **herança** e a **interação entre múltiplos atores** dentro de um cenário dinâmico.

## Estrutura do repositório
Para manter o projeto organizado, os arquivos foram divididos nas seguintes pastas:

* 📁 `codigos/`: Contém os arquivos de lógica do jogo (`.java`), as definições de contexto (`.ctxt`) e os arquivos compilados (`.class`).
* 📁 `images/`: Contém todas as texturas, sprites e backgrounds utilizados no cenário e nos atores.
* 📁 `sounds/`: Contém os efeitos sonoros e músicas de fundo que dão vida à experiência.

## Como executar
Como a estrutura de pastas original do Greenfoot foi modificada para fins de organização no repositório, para rodar o projeto localmente, é preciso seguir os passos abaixo:

### Pré-requisitos
* Ter o [Greenfoot](https://www.greenfoot.org/download) instalado em sua máquina.

### Passo a passo
1.  **Clone o repositório.**
2.  **Restaure a estrutura do Greenfoot:**
    O Greenfoot espera encontrar as classes, arquivos de configuração do projeto e as pastas `images/` e sounds/` todos compartilhando a mesma pasta raiz.
    * Entre na pasta `codigos/`.
    * Mova todos os arquivos contidos nela (`.java`, `.class`, `.ctxt`) diretamente para a pasta raiz do repositório (o mesmo nível onde ficam as pastas `images/`, `sounds/` e o arquivo `.greenfoot`).
3.  **Abra o projeto:**
    * Abra o Greenfoot.
    * Vá em **Project -> Open...** e selecione a pasta raiz do projeto.
4.  **Compile e jogue:**
    * Clique no botão **Compile** no canto inferior direito do Greenfoot.
    * Clique em **Run** para iniciar o jogo!

## Controles e mecânicas
**Objetivo:** Colete os fragmentos para liberar a passagem e avançar para a próxima fase. Desvie dos zumbis e das aranhas, pois eles causam dano.
* **[Setas do teclado]:** Movem a guardiã pelo cenário.
* **[Espaço]:** Atira os fragmentos (APENAS NA FASE 3).
* **[Enter]:** Inicia o jogo na tela de introdução.
